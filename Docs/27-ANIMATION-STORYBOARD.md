# EcoLoop — Animation Storyboard

Version: 1.0

Status: Final

Priority: High

---

# Purpose

This document defines all major motion sequences.

Animation should tell a story.

Not decorate the interface.

---

# Hero Sequence

Initial State:

```txt id="as01"
calm atmospheric field
```

---

After 2 seconds:

Subtle movement begins.

Particles drift.

---

User Moves Mouse

Particles react locally.

Field bends around cursor.

---

User Scrolls

Field compresses.

Density increases.

Transition into next section.

---

# Invisible Impact Section

Start:

Sparse particles.

---

During Scroll:

Hidden structures emerge.

---

End State:

Complex environmental network visible.

---

Meaning:

Invisible impact becoming visible.

# Dashboard Trend Change

Improving trend:

Field becomes:

* calmer
* more organized

---

Worsening trend:

Field becomes:

* denser
* more complex

---

Never:

```txt id="as04"
green good
red bad
```

# Empty State Storyboard

Initial:

Minimal field.

---

User interacts:

Field responds.

---

Prompt appears.

---

Encourages first action.

---

# Mobile Adaptation

Keep:

* narrative
* transitions

Reduce:

* particle count
* visual complexity

---

# Reduced Motion

Replace:

* particle movement
* field transitions

With:

* fades
* opacity transitions

---

# Definition of Done

Every animation should communicate:

* state
* progress
* understanding

Never animation for animation's sake.

End of Document.
