# EcoLoop — Carbon Field Specification

Version: 1.0

Priority: Critical

---

# Purpose

The Carbon Field is the visual representation of the user's environmental impact.

It is:

* dashboard centerpiece
* visual identity
* primary differentiator

---

# Core Concept

Users cannot see carbon.

The Carbon Field visualizes invisible environmental impact.

---

# Visual Metaphor

Think:

* weather system
* magnetic field
* atmospheric currents

Not:

* charts
* trees
* earth
