# EcoLoop Visual Direction

Version: 1.0

## Core Theme

Invisible Carbon

Users cannot see emissions.

EcoLoop visualizes invisible environmental impact.

---

## Emotional Goals

Users should feel:

* curious
* empowered
* intelligent
* optimistic

Users should never feel:

* guilty
* shamed
* lectured

---

## Visual Keywords

* atmospheric
* cinematic
* dimensional
* futuristic
* intelligent
* fluid

---

## Inspirations

Not:

* Greenpeace
* WWF
* Environmental NGOs

Instead:

* Linear
* Arc Browser
* Apple Vision Pro
* Stripe Sessions
* Nothing OS
* Raycast

---

## Core Metaphor

Carbon behaves like a flowing invisible substance.

Visualize:

* streams
* particles
* fields
* density
* flow

Never visualize:

* leaves
* trees
* earth
* forests

---

## Hero Direction

Full-screen immersive scene.

Three.js particle atmosphere.

Cursor movement subtly disturbs particle flow.

Particles represent environmental activity.

---

## Dashboard Direction

User footprint becomes a living structure.

Not charts.

Not cards.

The dashboard should feel like observing a system.

Charts support the experience.

They do not define it.
